---

## 🎮 游戏简介

M3 是一款 2D 多人在线角色扮演游戏，使用 Phaser 3 引擎开发。

### 核心特性

- 🗺️ 开放世界地图（50x50 网格）
- ⚔️ 实时战斗系统
- 🎒 装备系统（4 个槽位）
- 👾 怪物 AI（巡逻/Boss）
- 💻 可视化 UI（血条/蓝条/技能栏）

---
