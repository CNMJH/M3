# M3 游戏 - 快速开始指南

##  首次运行步骤

### 1. 克隆仓库后，首先安装依赖

```bash
cd Z:\ai\M3GIT
npm install
```

**这会下载所有需要的包（约 285 个），可能需要 1-3 分钟。**

### 2. 启动开发服务器

```bash
npm run dev
```

### 3. 打开浏览器访问

```
http://localhost:3001
```

---

## ⚠️ 常见问题

### 问题：`'vite' 不是内部或外部命令`
**原因**: 没有安装依赖
**解决**: 运行 `npm install`

### 问题：端口被占用
**解决**: 
```bash
# 查看占用端口的进程
netstat -ano | findstr :3001
# 杀死进程（替换 PID）
taskkill /PID <PID> /F
```

### 问题：Node.js 版本不兼容
**要求**: Node.js 18+ 
**检查**: `node -v`
**下载**: https://nodejs.org/
