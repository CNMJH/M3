## 📋 项目概况

M3 是一款 2D 多人在线角色扮演游戏，使用 Phaser 3 引擎开发。

### 核心特性

- 🗺️ **开放世界地图** - 50x50 网格，采集点，定时撤离点
- ⚔️ **实时战斗** - 靠近怪物自动战斗，技能系统
- 🎒 **装备系统** - 4 个装备槽，属性加成，技能绑定
- 👾 **怪物 AI** - 巡逻 AI，Boss 战
- 💻 **UI 系统** - HP/MP 血条，技能栏

## 📁 项目结构

```
M3/
├── src/
│   ├── game/
│   │   ├── core/scenes/      # 游戏场景
│   │   │   ├── BootScene.js  # 预加载
│   │   │   ├── MenuScene.js  # 菜单
│   │   │   └── GameScene.js  # 主游戏
│   │   ├── entities/         # 游戏实体
│   │   │   ├── player.js     # 玩家
│   │   │   └── monster.js    # 怪物
│   │   ├── systems/          # 游戏系统
│   │   │   ├── mapSystem.js  # 地图
│   │   │   ├── combatSystem.js # 战斗
│   │   │   └── network.js    # 网络
│   │   ├── ui/               # UI 界面
│   │   │   └── mainUI.js     # 主 UI
│   │   └── assets/config/    # 配置文件
│   └── index.js              # 入口
├── docs/                     # 文档
├── tests/                    # 测试
├── public/                   # 静态资源
├── index.html
├── package.json
└── vite.config.js
```

## 🚀 快速开始

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

访问：http://localhost:3001

### 构建

```bash
npm run build
```
